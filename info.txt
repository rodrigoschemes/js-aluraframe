Instalar o projeto Server (APIs)
- Node.JS
- cd server
- npm install

Instalar o projeto Client
- cd client
- npm install


Compilar o projeto
- cd client
- npm run build / npm run watch (compilar em tempo de execução)

Rodar o projeto
- cd server
- npm start
- http://localhost:3000
